## How to install
1. Clone this repository
   
   ```
   git clone https://github.com/thekevinkun/virtual-reality.git
   ```
2. Go into project
   
   ```
   cd virtual-reality
   ```
3. Install dependencies
   
   ```
   npm install
   ```
   
4. Run project
   
   ```
   npm run dev
   ```
